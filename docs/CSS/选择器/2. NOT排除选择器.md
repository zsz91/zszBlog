# NOT排除选择器
1.选择器可以链式操作 一直往后写
````
<html>
  <head>
    <!-- <link href="index.css" rel="stylesheet" /> -->
    <style>
      /* 取li里面的 奇数个 但是不取第3个 和 第1个 */
      main > ul li:nth-child(2n + 1):not(:nth-child(3)):not(:nth-child(1)) {
        color: red;
      }
    </style>
  </head>

  <body>
    <main>
      <ul>
        <li>第1个</li>
        <li>第2个</li>
        <li>第3个</li>
        <li>第4个</li>
        <li>第5个</li>
        <li>第6个</li>
        <li>第7个</li>
        <li>第8个</li>
      </ul>
    </main>
  </body>
</html>

````
