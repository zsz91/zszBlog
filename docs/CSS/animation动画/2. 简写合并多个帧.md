```
@keyframes name2 {
  25% {
    transform: scale(2);
    background-color: red;
  }
  50% {
    transform: scale(1);
    background-color: #e74c3c;
  }
  75% {
    transform: scale(2);
    background-color: blue;
  }
}
```
等价于以下代码 
```
@keyframes name2 {
    25% {
        background-color: red;
    }
    50% {
        transform: scale(1);
        background-color: #e74c3c;
    }
    75% {
        background-color: blue;
    }
    25%,75% {
        transform: scale(2);
    }
 
}
```
合并的写法不会覆盖单独定义的样式
