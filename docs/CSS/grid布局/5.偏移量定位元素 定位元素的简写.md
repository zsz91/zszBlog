#### 在元素中设置 

grid-column-end: span 3; //占3列

grid-row-end: span 2;  //占2行
````
  grid-row-start: 2;
  grid-row-end: span 1;
  grid-column-start: 2;
  grid-column-end: span 1;
  background-color: blueviolet;
````
#### 元素定位简写 推荐使用
grid-row: 2/3; 等价于 grid-row: 2/span 1;

grid-column: 2/3; 等价于 grid-column: 2/span 1;

等价于下面的代码
````
  grid-row-start: 2;
  grid-row-end: 3;
  grid-column-start: 2;
  grid-column-end: 3;
  特别注意 在less 中要 像下面这样写
  grid-row: ~"2/3";
  grid-column: ~"2/3";
````
