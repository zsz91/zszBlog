# line-height继承的坑
#### 写百分比 和写倍率 继承的line-height 计算方式并不同
1. 倍率: p标签自己的font-size * 倍率
2. 百分比 父级的 font-size * 百分比
```css
 body{
     font-size: 30px;
     line-height: 1.5; // p 标签 line-height = p fontSize * 1.5 = 20px
     line-height: 200%; // p 标签 line-height = body fontSize * 200% = 60px;
 }
 
 p{
     font-size: 12px;
 }
 <body>
 <p>内容</p>
 </body>
```
