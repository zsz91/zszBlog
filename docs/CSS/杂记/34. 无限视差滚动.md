# CSS-无限视差滚动
- [B站](https://www.bilibili.com/video/BV1Zz9dYwEs7)
- 监听鼠标滚轮滚动(wheel),添加合适的动画 class 到div上
- 监听动画效果结束(transitionend),移除对应的class动画并修改索引
```javascript
    window.addEventListener('wheel', (e) => {
        if(!e.deltaY) return;
        if(e.deltaY > 0) {
            console.log('> 0 向下滚动')
        } else {
            console.log('< 0 向上滚动');
        }
    })
```
```javascript
    document.addEventListener('transitionend', () => {
        console.log('动画结束回调');
    });
```
