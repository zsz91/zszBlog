#### CSS画三角形

```css
 .content{
             width:0;
             height: 0;
             border-right: 100px black solid;  
             /*如果希望不显示这个方向的三角形则设置成 transparent*/
             border-left: 100px blue solid;
             border-top: 100px yellow solid;
             border-bottom: 100px red solid;
         }
```
