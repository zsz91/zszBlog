# 实现数组 unshift 方法

```javascript
 Array.prototype.myUnshift = function () {
    let args = [...arguments];
    for (let i = args.length - 1; i >= 0; i--) {
        this.splice(0,0,args[i]);
    }
    return this.length;
}
console.log([1, 2, 3].myUnshift(4, 5, 6)); // 4,5,6,1,2,3
```
