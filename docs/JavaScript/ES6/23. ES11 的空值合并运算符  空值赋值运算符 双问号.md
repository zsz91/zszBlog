##### 空值合并运算符 ?? 出现原因
- 因为有时候我们想把假值如 0 和空字符串 "" 视为有效的数值。而逻辑或操作符不会返回假值。
```
alert(username ?? "Guest");  
等价于 =>  if(username === null || username === undefined) {
    return 'Guest'
}  
```

##### 空值赋值运算符 ??=
- 当 ??= 左侧的值为null,undefined的时候,才会将右侧变量的值赋值给左侧变量, 其他所有值都不会进行赋值
```
let b = '侬好';
let a = 0
let c = null;
let d = ’123‘
b ??= a;  // b = “侬好”
c ??= d  // c = '123'

```
