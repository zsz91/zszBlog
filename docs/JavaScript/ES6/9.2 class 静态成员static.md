## class静态成员
1. 特别注意 函数的name属性返回 一个函数声明的名称!!!
2. 子类不能继承父类的 静态属性
3. 子类可以重写父类的方法.
````
ES5中函数的静态成员
function Phone(){
    
}
Phone.name = '手机'; // 函数对象的属性 只属于函数对象 不属于实例对象
Phone.info = '手机的信息';
Phone.change = function(){
    console.log('我可以改变世界');
}

let nokia = new Phone();
console.log(nokia.name) // undefined name 不属于实例对象 
console.log(Phone.info) // 手机的信息
console.log(Phone.name) // Phone 这里不会输出手机 因为 函数的name属性返回 一个函数声明的名称!!!


````

```
ES6中class的静态成员写法 static
class Phone{
    static name = '手机';
    static change(){
        console.log('我可以改变世界');
    }
}
let nokia = new Phone();
console.log(nokia.name); // undefined
console.log(Phone.name); // 手机
```

## class的类继承

````
 class A{
    static name = '我是class A的静态属性'; // 子类无法继承
    publicProto = '我是class A的公共属性'; // 子类可以继承
 }

 let a = new A();
 console.log(A.name);  // 我是class A的静态属性. 
 // 函数式继承 和 class继承的区别之一. name属性
  console.log(a.name); // undefined
 console.log(a.publicProto); // 我是class A的公共属性
````
