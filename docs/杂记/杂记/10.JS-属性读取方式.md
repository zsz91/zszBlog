# ```obj.x``` 与 ```obj[x]``` 的区别
- 唯一的区别 是 ```obj[x]``` 会先判断x是不是Symbol 如果不是将 ```x``` 转为 ```String```
```javascript
const obj = {};

/**
 * [[Get]](obj, 'x', obj)
 */
obj.x;

/**
 * [[Get]](obj, x是不是Symbol ? x : String(x), obj)
 */
obj[x];
```

```javascript
const arr = [];
arr[0] = 1; // 
arr['0'] = 2;
console.log(arr); // 数组只有一项 0:2 且 length = 1
```
```javascript
const obj = {};
obj[{a: 1}] = 1;
obj[{a: 2}] = 2;
console.log(obj); // { '[object Object]': 2 }
```
