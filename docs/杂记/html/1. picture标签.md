# &lt;picture&gt; 标签
- 属于响应式相关的功能
- &lt;picture&gt; 元素用于定义多种图片资源，以便在多种分辨率的屏幕上显示最合适的图片。
- 可以理解为图像的“媒体查询”（Media Query）。

#### 示例 1：为不同屏幕尺寸提供不同图片

```html
<picture>
  <!-- 屏幕宽度 <= 768px 时使用小图 -->
  <source media="(max-width: 768px)" srcset="small.jpg">
  <!-- 屏幕宽度 > 768px 时使用大图 -->
  <source media="(min-width: 769px)" srcset="large.jpg">
  <!-- 后备图片 -->
  <img src="default.jpg" alt="风景">
</picture>
```
